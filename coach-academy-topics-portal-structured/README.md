# Coach Academy — Topics Portal (Static)

This repo is a **static, GitHub Pages–friendly** portal:

Home → Track → Session → Topic (videos/scripts).

## Run locally
Just open `index.html` in the browser (or use any static server).

## Edit content
- Main content: `data/portal-data.json`
- Instructor example profile content: `data/profile-data.json`

## Pages
- `index.html` (Home)
- `tracks/track.html?id=...`
- `sessions/session.html?track=...&session=...`
- `topics/topic.html?track=...&session=...&topic=...`
- `instructors/index.html`
- `instructors/assem-amr.html` (example profile page)
